# 🤖 Serverless AI Chatbot using Amazon Bedrock

A fully serverless AI chatbot built using AWS services.

## 🚀 Architecture
User → API Gateway → Lambda → Amazon Bedrock → Response

## 🛠 Tech Stack
- Amazon Bedrock
- AWS Lambda
- API Gateway
- Amazon S3
- HTML, CSS, JavaScript

## 📌 Features
- Serverless architecture
- AI responses using Titan Text model
- Static frontend hosted on S3
- REST API with API Gateway
- Secure IAM role access

## ⚙️ Setup Steps
1. Enable Amazon Bedrock
2. Create IAM role
3. Create Lambda function
4. Setup API Gateway
5. Deploy frontend to S3

## 🧹 Clean Up
Delete resources from:
- Lambda
- API Gateway
- S3
